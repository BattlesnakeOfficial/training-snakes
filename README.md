# Training Snakes

## Running it
Run the snake server locally:
```
> ./scripts/run_snake
```
this should provide all snakes at different paths, i.e.:
- http://localhost:8080/snake_1
- http://localhost:8080/snake_2
- http://localhost:8080/snake_3
- ...

Run the game server locally:
```
> ./scripts/run_server
```
